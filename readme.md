## Clone this repo

```
git clone git@github.com:fandredev/my-ml-guide.git
```

## Go to directory

```
 cd my-ml-guide
```

## Create your env using conda

```
 conda env create -f environment.yml
```

## Activate your env

```
 conda activate $YOUR_FOLDER_PATH/env
```
## Run jupyter [inside env]

```
 jupyter lab
```
or

``` 
jupyter nootbook
```
